# Pokedex fonctionnel codé en python

=======

## Consignes 
    *Créer une application Pokédex en utilisant Python et Tkinter. Cette application doit être capable de lister des Pokémon, afficher des informations sur un Pokémon spécifique lorsque l'utilisateur le sélectionne, et permettre à l'utilisateur d'ajouter de nouveaux Pokémon à la liste.

    Fonctionnalités de l'application
    Lister les Pokémon : L'application doit lister les noms des Pokémon existants. Cela peut être affiché dans une liste ou un menu déroulant.
    Afficher des informations sur un Pokémon : Lorsqu'un utilisateur sélectionne un Pokémon dans la liste, l'application doit afficher des informations sur ce Pokémon, comme son nom, son type, ses capacités, etc.
    Ajouter un nouveau Pokémon : L'application doit avoir une interface permettant à l'utilisateur d'ajouter un nouveau Pokémon à la liste. L'utilisateur doit être en mesure d'entrer le nom du Pokémon, son type, ses capacités, etc.
    Livrables
    Vous devez fournir le code sur ce site
    Vous devez versionner régulièrement votre code sur Github* -
